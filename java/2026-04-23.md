# 중첩 반복문
반복문은 반복문 안에 또 반복문을 넣을 수 있다.
```java
package loop;

public class Nested1 {
    public static void main(String[] args) {
        for (int i = 0; i < 2; i++) {
            System.out.println("외부 for 시작 i:" + i);
            for (int j = 0; j < 3; j++) {
                System.out.println("-> 내부 for " + i + "-" + j);
            }
            System.out.println("외부 for 종료 i:" + i);
            System.out.println();
        }
    }
}
```
위 처럼 반복문 안에 반복문을 넣어 외부 for문은 2번 내부 for문은 3번 실행 된다고 적혀있지만 외부 for문은 총 2번을 반복하기에 내부 for문이 2 * 3 = 6 6번 반복한다.

# 정리

## for문 장점
1. for문은 초기화, 조건문, 증감식을 한줄에 할 수 있다
2. 정해진 횟수만큼 처리하는 식에 쓰기 적합하다.
3. for문 안에서만 코드가 작동하므로 외부에서 건들기 힘들다.

## for문 단점
1. 조건이 루프 안에서 변경 될 경우 while이 더 적합하다.
2. 복잡한 조건문은 while문이 더 쉬울 수도 있다.

## while문 장점
1. 루프 안에서 조건식이 변경되는 경우 while문이 더 좋다
2. 더 복잡한 조건을 사용할 수 있다

## while문 단점
1. 초기화, 조건체크, 반복 후의 작업 등이 분산되어 있어 가독성이 떨어진다
2. 루프 변수가 while문 밖에서도 바꿀 수 있어 오류가 발생할 수도 있다.
