# 계산과 형변환

1. 같은 타입끼리는 같은 결과를 냄 int + int는 int를 double + double은 dobule을 나타냄
2. 만약 서로 다른 타입끼리 더하면 더 큰 자료형으로 바뀜 예로 int형 + double형은 double로 값이 자동으로 바뀜.

이러한 것으로 인해 나타나는 차이점은 int는 소수점자리 까지 나타내지 못하기 때문에 int a = 3; a / 2 하면 a는 1이 나오지만
double a = 3.0; a / 2 하면 a는 1.5가 나온다.

또한 위처럼 곤란한 문제가 생겼을 땐
int a = 3;
int b = 2;
double result = (double) a / b;
이렇게 하면 된다
