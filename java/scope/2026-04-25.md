# 스코프 존재 이유
변수를 선언한 시점부터 계속 사용해도 문제 없을 거 같은데 왜 지역 변수라는 개념을 만들었을까?

이유 첫 번째: 비효율적인 메모리 사용. 메모리 사용을 쓸데 없이 더 늘리게 된다.

이유 두 번째: 코드가 복잡해진다. 만약 if문안에 변수를 선언했으면 그 변수는 main에서 생각 안해도 되는데, main에서 변수를 선언하면
나중에 유지보수 할 때도 그 변수를 계속 신경써야 되기에 지역 변수를 만든 것이다.

또한 for문과 while문에서도 for는 지역 변수를 조건문에서 선언할 수 있기에 while문 보다 효율적인 것을 알게된다.
