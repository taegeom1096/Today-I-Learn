# 자동 형변환

일단 작은 범위에서는 큰 범위로 당연히 값을 넣을 수 있다.
ex) int -> long -> double (int < long < double)

큰 범위에서 작은 범위는 오버플로우, 소숫점 버림 현상이 나타날 수 있다.

형변환을 예로 들면
```java
package casting;

public class Casting1 {
    public static void main(String[] args) {
        int intValue = 10;
        long longValue;
        double doubleValue;

        longValue = intValue;
        System.out.println("longValue = " + longValue);

        doubleValue = intValue;
        System.out.println("doubleValue = " + doubleValue);

        doubleValue = 20L;
        System.out.println("doubleValue2 = " + doubleValue);
    }
}
```
위 같이 longValue = intValue라 지정할 때 자동으로 intValue는 long형으로 지정된다.
이런 것을 자동 형변환 또는 묵시적 형변환이라고 한다.
