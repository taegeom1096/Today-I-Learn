Scanner는 사용자에게 입력받는 클래스다.

Scanner를 사용하려면 가장 중요한 3가지        
1. import java.util.Scanner;                      
2. Scanner 변수명 = new Scanner(System.in);                                      
3. 자료형 변수명 = scanner.nextLine();                                

이거만 외워두고 있어도 Scanner정복이다.

그리고 설명하자면                                                        
1번은 Scanner 클래스를 불러오는 것이고                                                                
2번은 Scanner 클래스에서 System.in이라는 문자열을 읽어주는 것을 생성자 호출 해주는 것이고                                        
3번은 무슨 자료형태로 scanner에게 입력 받은 값을 변수에 저장하는 것이다. (참고로 문자열은 nextLine(), 정수는 nextInt(), 실수는 nextDouble()임)                 

```java
package scanner;

import java.util.Scanner;

public class Scanner1 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("문자열을 입력하세요:");
        String str = scanner.nextLine();
        System.out.println("입력한 문자열: " + str);

        System.out.print("정수를 입력하세요:");
        int intValue = scanner.nextInt();
        System.out.println("입력한 정수: " + intValue);

        System.out.print("실수를 입력하세요:");
        double doubleValue = scanner.nextDouble();
        System.out.println("입력한 실수: " + doubleValue);

    }
}

```
앞에서 말한 중요한 3가지를 활용한 예시 코드
