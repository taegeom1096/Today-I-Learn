# 상속
오늘은 상속에 대해 알아보겠다.                      
상속이란 기존 클래스(부모)의 변수와 메서드를 새로운 클래스(자식)가 그대로 물려받아 재사용하는 문법 요소라고 한다.
```java
// 부모 클래스
class Animal {
    void breathe() {
        System.out.println("숨을 쉽니다.");
    }
}

// 자식 클래스 (extends 키워드 사용)
class Dog extends Animal {
    void bark() {
        System.out.println("멍멍!");
    }
}
```
이렇게 나타낼 수 있다 즉 Animal을 Dog한테 상속시킨다는 것이고 Dog가 breathe()를 사용할 수 있다는 것.

또한 상속 종류 중 하나 더 있는데 바로 오버라이딩(재정의)이다 이것은 부모한테 상속받은 것을 자기의 입맛대로 바꾸는 것이다.
예로 Dog가 Animal한테 상속 받았는데 멍멍이라는 것을 내가 왈왈로 바꾸는 것이다.
