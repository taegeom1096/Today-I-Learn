# 1. 반복문 시작

반복문은 특정 코드를 반복한다

자바는 3가지의 반복문을 지원한다.

while, do-while, for

반복문은 while만 있어도 가능하지만 더 쉽게 코드를 짜기 위해서 do-while, for도 사용한다.

# 2. while문1

while문은 조건식을 확인하고 그 조건식이 참이면 코드를 실행하고, 거짓이면 코드를 빠져나간다.
이후 코드가 끝나면 다시 조건식을 검사한다.

만약 while (count < 3) 이라는 식이 있고 그 안에 count++있으면, count가 3이 될 때까지 반복하다가 count가 3이 되면 이 반복문은 끝이 난다.

# 3. while문2

```java

package loop;

public class While2_2 {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1;
        sum = sum + i;
        System.out.println("i=" + i + " sum=" + sum);
        i++;
        sum = sum + i;
        System.out.println("i=" + i + " sum=" + sum);
        i++;
        sum = sum + i;
        System.out.println("i=" + i + " sum=" + sum);
        i++;
    }
}

```

위와 같은 whille문을 사용하면 매1우 비효율적이다.

but 아래와 같은 코드는

```java
package loop;

public class While2_3 {
    public static void main(String[] args) {
        int sum = 0;
        int i = 1;
        int endNum = 3;
        while (i <= endNum) {
            sum = sum + i;
            System.out.println("i=" + i + " sum=" + sum);
            i++;
        }
    }
}
```

while에 조건을 넣어서 매우 단축시킬 수 있다. 

i는 endNum보다 클때 반복문이 멈춘다.
