# 4대 특징

## 1. 캡슐화 

- 정의: 관련 있는 데이터와 메소드를 하나의 클래스로 묶고, 외부에서 내부 구현 내용을 숨기는 것. -> 정보은닉
- 목적: 데이터 보호 및 외부 변경에 대한 영향 최소화.
- 핵심: private 변수와 getter/setter 메소드를 통해 데이터에 접근.

즉 필요한 부분만 외부에 공개하는 방식으로, 노출될 필요가 없는 내부 구현 정보를 숨김.

```java
public class Person {
    private String name; // 외부 접근 불가 (private)
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // getter/setter를 통해 안전하게 데이터 접근
    public String getName() { return name; } // 외부에서 읽기만 가능
    public void setAge(int age) { if(age > 0) this.age = age; } // 데이터 검증 및 저장
}
```

## 2. 상속화

- 정의: 상위 클래스(부모)의 특징을 하위 클래스(자식)가 물려받아 재사용하거나 확장하는 것.
- 목적: 코드의 재사용성을 높이고, 중복을 줄이며, 클래스 간 계층 구조 확립.
- 핵심: 부모의 기능을 자식이 재정의(Overriding)하여 사용.

```java
// 부모 클래스
class Animal {
    void eat() { System.out.println("먹는다"); }
}

// 자식 클래스 (Animal 상속)
class Dog extends Animal {
    void bark() { System.out.println("멍멍"); }
}
```

## 3. 다형성

- 정의: 하나의 객체나 메소드가 여러 가지 형태를 가질 수 있는 능력.
- 목적: 코드의 유연성을 높이고, 인터페이스를 통해 다양한 객체 구현 가능.
- 핵심: 오버로딩(Overloading - 같은 이름의 메소드 매개변수 다르게)과 오버라이딩(Overriding - 부모 메소드 재정의).

```java
class Cat extends Animal {
    @Override
    void eat() { System.out.println("생선을 먹는다"); } // 오버라이딩
}

Animal myPet = new Cat(); // 부모 타입으로 자식 객체 참조
myPet.eat(); // "생선을 먹는다" 출력 (다형성)
```

## 4. 추상화

- 정의: 복잡한 내부 구현은 숨기고, 핵심적인 공통 특징(본질)만 추출하여 정의하는 것.
- 목적: 모델링을 단순화하여 이해하기 쉽고, 변경에 유연한 설계 가능.
- 핵심: abstract class(추상 클래스)나 interface(인터페이스)를 사용하여 구현의 자유도를 높임.

```java
public abstract class Menu {

    void showMenu(){}
}

class Coffee extends Menu {
    private String name;
    private int price; 
    
    @Override
    void showMenu() {
        System.out.println(name);
        System.out.println(price);
    }
}

class Latte extends Menu {
    private String name;
    private int price;

    @Override
    void showMenu() {
        System.out.println(name);
        System.out.println(price);
    }
}

class Juice extends Menu {
    private String name;
    private int price;

    @Override
    void showMenu() {
        System.out.println(name);
        System.out.println(price);
    }
}
```
