# break, continue

break는 만약 while문이 있는데 거기에 그 while 문 코드 안에 break가 있으면 그 뒤에 코드들은 실행 되지 않고 while문은 종료된다.
continue는 while문 안에 있는 코드 중 continue 뒤에 코드들을 실행하지 않고 다시 조건식으로 이동한다. 그리고 조건식이 참이면 while문을 다시 실행한다.

# for문1
for문은 for(초기식; 조건식; 증감식) 으로 구성되있다. 실행 순서는
1. 초기식을 실행한다. 초기식은 한 번만 사용된다.
2. 조건식을 검증한다. 조건식이 참이면 코드를 실행하고, 거짓이면 for문을 빠져나간다.
3. 코드 실행
4. 코드가 끝나면 증감식을 실행한다.
5. 반복(초기식 빼고)

또한 for문과 while문의 차이는 for문이 초기식, 조건식, 증감식을 정리할 수 있기에 더 간편한 것이 차이점이다.
# for문2
for(; ;)는 초기식과 조건식 증감식이 없으므로 while문과 똑같이 무한반복을 한다.
즉 for문은 모든 반복을 수행 할 수 있다.
